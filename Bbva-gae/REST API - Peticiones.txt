El objetivo de este documento es explicar el funcionamiento de la REST API.


Se pueden hacer peticiones tanto a nivel AJAX, como librerias propias de Google.

LA URL DEL Servicio:
https://sopragroupux.appspot.com/_ah/api/discovery/v1/apis/evento/v4/rest



URL Peticiones AJAX.

Listar Event 
	URL: https://sopragroupux.appspot.com/_ah/api/evento/v4/event
	httpMethod: "GET"

Insertar Evento
	URL: https://sopragroupux.appspot.com/_ah/api/evento/v4/event
	httpMethod: "POST"
	data: jSON de la estructura de Event.

Eliminar Evento
	URL: https://sopragroupux.appspot.com/_ah/api/evento/v4/event/{id}
	httpMethod: "DELETE"
	data: {id} del elemento a eliminar
